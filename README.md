# ✅ TaskFlow — Task Management App

A full-featured task management app built with HTML, CSS, and JavaScript. Designed with a sidebar layout, stats dashboard, and productivity-focused features.

## 🌐 Live Demo
[View Live →](https://shahzaibuxui-sketch.github.io/taskflow)

## ✨ Features

- **Task Categories** — organize tasks by Personal, Work, Shopping, Health, Education
- **Priority Levels** — mark tasks as Low, Medium, or High priority
- **Due Dates** — set deadlines with overdue detection
- **Stats Dashboard** — live count of total, completed, pending tasks and completion rate
- **Search** — instantly filter tasks by keyword
- **Sidebar Navigation** — filter between All, Active, and Completed tasks
- **Dark Mode** — toggle with preference saved to localStorage
- **Persistent Storage** — all tasks saved locally, survive page refresh
- **Bulk Actions** — clear completed tasks or delete all at once
- **Notifications** — toast messages for task actions
- **Fully Responsive** — works on all screen sizes

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, CSS Variables, Animations)
- Vanilla JavaScript (localStorage, DOM Manipulation, Date API)
- Google Fonts — Inter
